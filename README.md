# North Maple Immigration Consulting Landing Page

Single-page React website for a solo incorporated RCIC (Canadian Regulated Immigration Consultant) practice.


## Run locally (development)

```bash
npm install
npm run dev
```

Then open the local URL shown by Vite (usually `http://localhost:5173`).

## Production build

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```

Then open the local URL shown by Vite preview (usually `http://localhost:4173`).

> Note: `npm run build` only creates files in `dist/`; it does not automatically open a browser.
